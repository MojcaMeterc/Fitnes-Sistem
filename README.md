# Fitnes-Sistem
Sistem omogoča, da uporabniki (člani fitnesa) rezervirajo termine vadb pri različnih trenerjih v različnih dvoranah. Vsak termin ima določen tip vadbe, datum in uro. Rezervacija povezuje uporabnika in termin. Uporabniki lahko tudi kupijo karte ter zabeleženo je do kdaj traja in kakšne vrste karto imajo ter koliko karta stane.
